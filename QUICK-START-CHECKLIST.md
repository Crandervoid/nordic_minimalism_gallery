## ðŸ“… Days 4-6: Build (6 hours, 2hr per sprint)

### Sprint Pattern (repeat 3x)

**Start:**
- [ ] Review sprint goals
- [ ] Have reference images ready

**Build:**
- [ ] Describe changes to AI
- [ ] AI generates code
- [ ] Take screenshot
- [ ] Compare to reference
- [ ] Iterate 2-3 times

**Review:**
- [ ] Run Lighthouse
- [ ] Check mobile responsive
- [ ] Test accessibility
- [ ] Review against style guide

**Document:**
- [ ] Save all screenshots
- [ ] Note what worked/didn't
- [ ] Track iterations
